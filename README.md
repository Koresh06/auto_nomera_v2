# auto_nomera_v2

Telegram-бот для размещения объявлений о продаже/покупке автомобильных номеров.

## Стек

- Python 3.12, aiogram 3, aiogram-dialog
- SQLAlchemy 2.0 async + PostgreSQL
- Redis (холд слотов, очередь задач)
- Taskiq (воркер + планировщик)
- dishka (dependency injection)
- uv (управление зависимостями)

## Архитектура

Clean Architecture + DDD: `domain → application → infrastructure → presentation`

## Запуск

```bash
# зависимости
uv sync

# миграции
alembic upgrade head

# бот
python -m src.presentation.telegram.app

# воркер (в отдельном терминале)
taskiq worker src.worker:broker

# планировщик (в отдельном терминале)
taskiq scheduler src.scheduler:scheduler
```

## Функциональность

### Объявления

- Продажа, покупка, срочный выкуп, магазин номеров
- Повторная публикация без повторного ввода данных
- Редактирование до и после публикации (номер, город, цена, телефон)
- При смене номера автоматически перегенерируется изображение и обновляется в канале
- Лимиты: 5 публикаций за 7 дней, интервал 7 дней для одного номера (настраивается по региону)

### Срочный выкуп (URGENT_BUYOUT)

- Отдельный укороченный флоу без выбора слота и города: номер → фото → цена → подтверждение
- Заявка уходит на модерацию администраторам с фото и кнопками "Одобрить" / "Отклонить"
- Двойная защита от race condition: если заявку уже обработал другой админ, второй получает уведомление об этом
- После одобрения статус объявления меняется на `PUBLISHED`, но публикации в канал не происходит — объявление становится доступно в каталоге раннего доступа
- После отклонения — статус `ARCHIVED`
- Рассылка пользователям с активной подпиской раннего доступа выполняется асинхронно через отдельную задачу в Taskiq (чтобы не блокировать модератора при большом числе подписчиков)

### Каталог раннего доступа (PRE_PUBLICATION)

- Доступен пользователям с активной подпиской `pre_publication_expires_at`
- Объединяет два источника:
  - все одобренные заявки на срочный выкуп (`URGENT_BUYOUT`, статус `PUBLISHED`)
  - все обычные публикации (продажа/покупка/магазин), которые выйдут в канал в ближайшие N часов (по умолчанию 2 часа до `publish_at_utc`)
- Сортировка: новые объявления — в начале списка
- Карточка объявления рендерится через единый `AdTextRenderer` — тот же текст, что увидят в канале, плюс пометка типа записи (🚨 срочный выкуп / 🕐 время до публикации)
- Кастомная пагинация по одной карточке с кольцевой навигацией (с последней страницы — на первую и обратно)
- Список всех позиций отдельным окном — тап по строке сразу открывает нужную карточку
- Удаление объявления администратором с подтверждением одним нажатием; если удаляется ещё не вышедшая публикация — соответствующая задача снимается с планировщика и публикация помечается `CANCELED`
- Фильтрация по региону — пользователь видит только объявления своего региона

### Слоты

- Календарь слотов с холдом на 15 минут (Redis)
- Перед финальным подтверждением повторно проверяется, что холд не истёк; если истёк — пользователя возвращает на календарь с уведомлением
- Платные и бесплатные слоты
- Системные платные слоты (настраиваются по региону)
- Конвертация холда в постоянную запись о занятом слоте (PostgreSQL) после подтверждения

### Платные услуги (паттерн Стратегия)

- **HIGHLIGHT** — красная рамка на фото объявления (наложение через Pillow)
- **PIN** — закрепление публикации в канале на N дней с автоматическим откреплением по таймеру
- **AUTOPUBLISH** — автопубликация объявления N дней подряд без повторных действий пользователя
- **PRIORITY_PUBLISH** — снять с очереди и опубликовать немедленно
- **PRE_PUBLICATION** — подписка на доступ к каталогу раннего доступа

### Оплата

- Архитектура платежей готова: `Payment` entity, идемпотентный `ConfirmPaymentUseCase`, обработка всех видов покупок (баланс, услуги, подписка, слоты)
- Подключение реальных платёжных провайдеров (ЮKassa, криптовалюта, Telegram Stars) в разработке — сейчас подтверждение оплаты выполняется как мок для тестирования флоу

### Регионы

- Мультирегиональность с отдельными каналами
- Таймзоны, настройки слотов, лимиты публикаций — всё настраивается per-region
- Ссылки на чат, VK-группу, канал в MAX подключаются индивидуально для каждого региона и подставляются в текст объявлений
